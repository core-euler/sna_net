# JungAI v0.3 — Public Release Notes

Date: 2026-03-13

## What's New

- Automatic dream analysis after saving.  
  You no longer need to start analysis manually: the dream is saved immediately and analyzed in the background.

- New dream grid view (3-column layout).  
  The main screen is now more compact and easier to scan.

- Improved dream cards.  
  Updated title/date presentation, better readability, and clearer analysis states.

- Updated onboarding flow.  
  A multi-step modal onboarding with a softer, more guided first-time experience.

- Voice input for dream capture (MVP).  
  Local audio recording with transcription into text.

- Dream Map.  
  A new map screen with symbolic nodes, archetype filters, detail view, and manual refresh.

- Bottom navigation menu.  
  Main sections are now accessible through a footer with icons.

## Improvements

- Better stability for background analysis and error handling.
- Clearer loading behavior for long-running analysis tasks.
- Markdown rendering support in analysis and chat responses.
- Improved responsiveness on real mobile devices.

## Important Behavior Changes

- New dream analysis starts automatically after save.
- Dream Map now uses a bounded map area (no infinite cyclic scrolling).
- Map filters now expose the full set of available user archetypes.

## How to Update

1. Download the APK from GitHub Release.
2. Install it over your current app version.
3. Open the app and let it sync data.

## Note

If you have dreams created in older versions, some new map elements may appear progressively after re-analysis and data refresh.
